# Overlay React App

Independent React application for the Electron overlay window.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Starts Vite dev server on http://localhost:5173

## Build

```bash
npm run build
```

Builds the app to `dist/` directory.

## Dependencies

- React 18
- TipTap editor with Yjs collaboration
- Vite for building
